# visvoai

*VisvoAI™*

`pip install visvoai` — the friendly name for the
[VisvoAI CLI](https://pypi.org/project/visvoai-cli/), a terminal coding agent
that takes permissions seriously. This meta-package only depends on
`visvoai-cli`; all code and docs live there.
